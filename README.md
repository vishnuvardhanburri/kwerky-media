# Kwerky Media

A high-performance React landing page for Kwerky Media, built with modern web technologies and tailored for a premium dark-mode storytelling experience.

## 🚀 Technologies

- React 19
- TypeScript
- Tailwind CSS
- React Suspense
- Code Splitting

## ⚙️ Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
